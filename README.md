# python-file-organizer
A Python-based file organizer that automatically sorts files by extension to improve directory management.

## Features
- Automatically sorts files into folders based on file type
- Skips directories and files without extensions
- Simple and easy to use

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Run the script using Python
3. Enter the path of the directory you want to organise
