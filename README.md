# Data Classification & File Segregation Tool

A Python-based automation tool that classifies and organizes heterogeneous data files into structured directories based on file type and data format.


## Features
- Segregates tabular, structured, log, media, and document data
- Handles mixed datasets automatically
- Easily extensible classification logic
- Simple and easy to use

## Technologies Used
- Python
- OS and file system handling

## How to Run
1. PLace mixed data files into a folder
2. Run the script using Python
3. Enter the path of the directory you want to organise
